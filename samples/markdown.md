## Common Programming Symbols
These symbols should render correctly in monospace fonts:

**Operators**: `+ - * / % = == != < > <= >= && || ! & | ^ ~ << >>`

**Brackets**: `() [] {} <>`

**Punctuation**: `; : , . ? ! " ' \` @ # $ % ^ & *`

**Special**: `→ ← ↑ ↓ ∞ λ α β γ δ ε θ π σ φ ψ ω`

**Box Drawing**: `┌┐└┘├┤┬┴┼│─`

**Arrows**: `⟶ ⟷ ⟵ ↗ ↘ ↙ ↖ ⇒ ⇔ ⇐`
